# YCA Cerámica 🪴

Sitio web artesanal para **YCA Cerámica**, un espacio dedicado a la cerámica hecha a mano: piezas únicas, cursos y herramientas para ceramistas.

🌐 **[ycaceramica.github.io](https://ycaceramica.github.io)**

---

## ¿Qué ofrece este sitio?

- **Productos** — Piezas cerámicas artesanales como mates, cuencos y tazas esmaltadas
- **Herramientas** — Calculadora de yeso para calcular proporciones de mezcla
- **Cursos** — Desde introducción a moldes hasta producción cerámica avanzada
- **Galería** — Fotos del taller y las piezas
- **Contacto** — Redes sociales y WhatsApp directo

---

## Características técnicas

- ✅ Diseño responsivo (funciona en celular y computadora)
- ✅ Modo oscuro / claro con memoria (se recuerda tu preferencia)
- ✅ Animaciones suaves al hacer scroll
- ✅ Calculadora de yeso integrada con proporciones para moldes rectangulares y circulares
- ✅ Botón de WhatsApp flotante
- ✅ Menú hamburguesa en móvil

## Calculadora de yeso

Herramienta pensada para ceramistas que trabajan con moldes. Permite calcular la cantidad exacta de agua y yeso según:

- Tipo de molde (rectangular o circular)
- Proporción de mezcla (70%, 75% o 80%)
- Cantidad de moldes a preparar
