El punto de equilibrio te dice cuántas piezas necesitás vender para no perder dinero. A partir de esa cantidad, cada pieza vendida es ganancia pura.

  1. Costos fijos — gastos que pagás siempre, aunque no vendas nada: alquiler, horno, herramientas, etc.
  2. Costo variable por pieza — lo que gastás para hacer cada pieza. Podés traerlo desde la Calculadora de Costos.
  3. Precio de venta — el precio al que vendés cada pieza. También puede venir de Costos.
  4. Simulador — mové el slider para ver cuánto ganás o perdés según cuántas piezas vendés.
