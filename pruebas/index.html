🧪 Generador de Pruebas

Generá series de pruebas de colorantes automáticamente y organizá tus testeos con precisión.

Generá hasta 50 pruebas de una sola vez Descargalas en PDF para llevar al taller

El generador crea una serie sistemática de pruebas variando uno o dos colorantes sobre una receta base fija.

  1. Receta base — ingresá los materiales de tu esmalte o engobe base (sin colorantes).
  2. Colorantes a variar — elegí uno o dos colorantes y el rango de porcentajes a testear.
  3. Tamaño de prueba — cuántos gramos vas a preparar por prueba.
  4. Generá la serie — la calculadora crea todas las combinaciones con los gramos exactos de cada ingrediente.

El resultado es una planilla lista para llevar al taller. Cada prueba tiene un código único para identificarla una vez cocida.
